Mutable Variables

A variable in OCaml is immutable, you will never be able to change the value associated with x (you get an error):

x := x + 1


So to get a traditionnal mutable variable, you must use references:

let x = ref 42


You can then change the value of x

x := 56


! allows you to access the contents of the mutable variable:

let y = !x + 1
